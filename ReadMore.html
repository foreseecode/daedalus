A Changing Climate in Data Privacy

Online privacy concerns have hit the mainstream media. These reports are impacting major policy changes like GDPR, and the public perception of data tracking & privacy. We believe the uptrend in users opting-in to use new tools to block trackers ensuring data privacy and protection will continue and will have significant implications for the web analytics & tracking business. We have created a data-less tracker that won't be blocked by browsers or filter lists and is 100% GDPR compliant.

Go Here for a live Demo of the DaedalusTracker: www.daedalus.live

Data Captured vs Data Transferred

We observe all user activity within the browser. Clicks, mouse moves, text typed, forms submitted, items bought.

Deploying TenserFlow Light within the tracker allows the tracker to use AI & ML models to interpret user actions and patterns in real time to define user behaviors.

The tracker only sends aggregated, and computed data about a user observed within the browser, the Persona. No raw user data or IDs are sent and saved on a server.

Personas with Behavioral Context

The personas are built from the many combined permutations of the user behaviors, or dimensions. Configuring these personas would be available for clients.

On the server side, and within the CXSuite a persona can be enumerated or aggregated to enrich analysis across the CXSuite features set.

Acting on Personas in Real-Time

Active personas are sent to the server and reported on in the CXSuite in real-time.

The stream of personas can control the deployment of Contextual Surveys, allowing for more engaging real-time interactions with users.
